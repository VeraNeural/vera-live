## 1. PURPOSE

Define exact responsibilities, permissions, and prohibitions for:

- Neural (cognitive intelligence)
- VERA (unified user-facing intelligence)

This contract exists to prevent layer collapse, role bleed, and silent drift.

## 2. LAYER DEFINITIONS (NON-NEGOTIABLE)

- Detect intent
- Detect adaptive codes
- Detect loops and avoidance patterns
- Generate:
  - observations
  - hypotheses
  - structured reasoning
  - candidate insights (flagged, conditional)
  - suggested response types (not wording)

Neural MAY NOT:

- Generate user-facing prose
- Decide tone, pacing, or depth
- Regulate emotion
- Apply challenge without IBA approval
- Override state-based safety

- Reframe without distorting meaning
- Reduce, soften, or delay insights

- Preserve agency and coherence

- Invent insights not present in Neural output
- Escalate depth beyond IBA limits
- Introduce challenge if blocked
- Override IBA governance

- Force depth = light
- Override model selection
- Trigger safety fallback
- Suppress Neural outputs entirely

IBA MAY NOT:

- Generate language
- Interpret meaning
- Engage with user content directly
- Optimize for performance, engagement, or cleverness

IBA always chooses restraint over brilliance.

## 3. INTERACTION RULES (ENFORCEMENT)

### Rule A ‚Äî One-way authority

IBA ‚Üí governs Neural + VERA

Neural ‚Üí informs VERA

VERA ‚Üí speaks to user

No upward overrides. Ever.

### Rule B ‚Äî Output ownership

Neural owns analysis

VERA owns language

IBA owns permission

If any layer crosses ownership ‚Üí violation.

### Rule C ‚Äî Safety override

If IBA blocks:

- Neural must stop generating insight
- VERA must reduce depth
- Model selection must fall back to safety

### Rule D ‚Äî Ambiguity handling

If:

- Signals conflict
- Confidence is low
- Layers disagree

Then:

- IBA forces VERA lead
- Pace = slow
- Depth = light
- Challenge = none

## 4. FAILURE CONDITIONS (HARD FAIL)

The system is out of contract if:

- Neural language appears verbatim to user
- VERA introduces new analysis not present in Neural
- Challenge occurs without IBA approval
- Tone shifts reveal internal routing
- Any layer references models or internal logic
- State safety is overridden by user request

Any of these ‚Üí block deploy.
